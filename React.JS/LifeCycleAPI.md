# Life Cycle API
## Life Cycle API 소개 및 사용
- Component가 나타나고 업데이트되고 사라질 때 이 중간 중간의 과정에서 특정한 작업을 하고싶을 때
- **Life Cycle**
  - Mounting : component가 Browser에 나타나는 과정
  - Updating : component의 state나 props가 바뀌는 과정
  - Unmounting : component가 Browser에서 사라지는 과정
