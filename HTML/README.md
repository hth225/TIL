## DTD (DOCTYPE)
- <!DOCTYPE html>
- DTD (Document Type Definition)
- 브라우저로 하여금, 문서의 정의를 알림

## 띄어쓰기
- &nbsp;

## 목록
- 연관되어 있는 항목들을 나열할 때 사용
- 순서가 없는 항목들은 ul(unordered list), 순서가 없는 항목은 ol(ordered list)를 사용
- css와 함께 메뉴로도 사용

순서가 없는 항목
> <ul>
>   <li>항목</li>
> </ul>
순서가 있는 항목
> <ol>
>   <li>항목</li>
> </ol>
